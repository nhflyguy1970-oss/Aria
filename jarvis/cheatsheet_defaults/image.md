# Image generation — Quick Reference

Stored in memory namespace **`cheatsheet`** (key: `image`). Edit in the Memory tab or say **"image cheatsheet"**.

## Chat examples

| You say | ARIA does |
|---------|-------------|
| "Generate an image of …" | ComfyUI / local generation |
| "Make it more detailed" | Prompt refinement follow-ups |

## Settings

- ComfyUI checkpoint in sidebar: SDXL quality / flux / etc.
- GPU vs CPU mode for ComfyUI when VRAM is tight.

## Gallery

- Generated images appear in the **Gallery** tab.

## CLI

```bash
python main.py image
```
