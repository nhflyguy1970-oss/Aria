# Aria Operational Improvement Log

Quiet evidence log for production operation.  
Do not interrupt the user about entries here.  
Review only on request or at a scheduled monthly operations review.

**Schema per observation**

| Field | Meaning |
|-------|---------|
| Date | ISO date of observation |
| Subsystem | Affected area (e.g. Mission Control, Journal, Memory) |
| Workflow | What the user was trying to do |
| Evidence | Concrete observation (error, friction, timing, repeated action) |
| Frequency | once / occasional / repeated / chronic |
| Severity | low / medium / high / critical |
| Impact | Effect on trust, time, or correctness |
| Category | Bug · Security · Performance · Workflow · Usability · Automation · Documentation · Maintenance · Research |
