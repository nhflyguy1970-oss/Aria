# Daily Use Mode

Aria Core V2 Alpha architecture is **complete**. Do not add organs, buses, or cognitive layers unless a blocking Architecture Problem Report exists.

## Rule

Development is driven only by real daily usage, production bugs, measured bottlenecks, verified usability issues, and repeat friction.

Every task begins with a **Problem Report** (see Mission Control **Product Backlog** on AI Platform).
